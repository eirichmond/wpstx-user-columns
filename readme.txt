Custom columns in the Users List

== Description ==

This plugin will add a custom column pulled from the saved user meta by users ID and display a value in the list
of users listed under Users from the main WordPress dashboard.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload wpstx-user-columns.zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
